Hello!
1. After pressing play you will be asked for a max calculation value. For example, if you select 10 you will only be given calculations with values below 10.
2. For each correct answer, one second will be added to the timer (10 seconds max). If the timer hits 0, game over!
3. For divisions where the answer would typically have a decimal point the correct answer is the value without the remainder.
4. You get 1 point times your max value per correct answer.
Good luck!
